# GlowGlam AI

GlowGlam AI is a full-stack AI-powered skincare platform that provides smart skin analysis, product recommendations, and an AI skincare chatbot.

## Project Overview

The platform allows users to:

- create an account and log in
- upload a facial image for skin analysis
- receive AI-powered skincare insights
- chat with an AI skincare assistant
- explore skincare products and brands
- manage analysis history and saved products in a dashboard

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- NestJS
- TypeScript
- REST API

### Database
- PostgreSQL
- Prisma ORM

### AI / Image Processing
- LLM API for chatbot
- MediaPipe Face Detection for facial validation

### Cloud / Storage
- AWS S3 for image storage
- AWS Amplify for frontend hosting
- PostgreSQL / RDS for database
